# a minesweeper game
a totally self-coded MFC application, running on Windows.

## how to run it?
1. you need to install __visual studio 2017__ or later version, and select __MFC__ devolopment while installing.
2. use __git clone__ to clone this repository to your own devolopment enviroment.
3. now open __MineCraft.sln__ file in the project folder with visual studio and run the code.
